## About XeeonTV

**XeeonTV** is a premium, modern IPTV and media consumption application designed for Android. It seamlessly organizes your Live TV, Movies, and Series into a beautiful, intuitive interface. XeeonTV automatically enriches your media library with rich metadata, posters, and backdrops to provide a top-tier viewing experience.

*Note: This repository is for distributing official releases, tracking issues, and gathering user feedback. The source code is closed and proprietary.*

## ✨ Features

- **Rich Media Library:** Automatically categorizes your m3u playlists into Movies, Series, and Live TV.
- **TMDB Integration:** Merges your playlists with TMDB to fetch gorgeous posters, backdrops, ratings, and cast information.
- **Advanced Video Player:** Powered by ExoPlayer, featuring:
  - Picture-in-Picture (PiP) support
  - Swipe gestures for Brightness and Volume control
  - Aspect Ratio and Orientation locks
  - Subtitle and Audio track selection
- **Continue Watching:** Automatically tracks your watch progress across all your movies and TV episodes.
- **Dynamic Theming:** Built with Jetpack Compose Material 3. Includes beautiful customized palettes (like Slate) and Dark mode by default.
- **Smart Discoverability:** Browse Trending, Popular, and Latest items easily with Netflix-style carousels.

## 📥 Installation

### Latest Release
You can download the latest version of XeeonTV from the **[Releases page](https://github.com/ZUBAYRALY/XeeonTV/releases)**.

1. Go to the [Releases](https://github.com/ZUBAYRALY/XeeonTV/releases) tab.
2. Download the latest `xeeon-tv-release-vx.x.x.apk` file under the **Assets** section.
3. Transfer the APK to your Android device (if downloading from a PC).
4. Tap the APK file to install. *(You may need to enable "Install unknown apps" from your phone's settings).*
